# Danh Sách Câu Hỏi và Hình Ảnh Cần Thiết

## 📌 Hướng Dẫn
- Mỗi câu hỏi sẽ cần 1 hình ảnh
- Đặt tên hình ảnh theo ý nghĩa (tiếng Anh hoặc ký tự có ý nghĩa)
- Hình ảnh sẽ được upload vào thư mục: `app/src/main/res/mipmap-hdpi/`
- Định dạng hình ảnh: PNG, JPG (khuyến nghị 400x300px)
- **Tên file không được chứa khoảng trắng hoặc ký tự đặc biệt**, chỉ dùng chữ cái, số, gạch dưới

## 🎯 **BƯỚC TIẾP THEO**

1. **Thu thập hình ảnh**: Tìm 100 hình ảnh phù hợp từ các nguồn:
   - Canva, Freepik, Unsplash, Pixabay (free images)
   - Pinterest, Google Images
   - Vẽ hoặc tạo hình ảnh của riêng bạn

2. **Đặt tên hình ảnh**: Sử dụng tên file đề xuất ở trên

3. **Chuẩn bị hình ảnh**: 
   - Kích thước: 400x300px (hoặc lớn hơn)
   - Định dạng: PNG hoặc JPG
   - **Lưu ý**: Tên file không được chứa khoảng trắng

4. **Upload vào dự án**:
   - Copy tất cả 100 hình ảnh vào `app/src/main/res/mipmap-hdpi/`

5. **Báo lại cho tôi**: Sau khi upload xong, tôi sẽ:
   - Cập nhật Question model để sử dụng drawable resources
   - Update LessonDetailActivity để load hình từ mipmap
   - Cập nhật Firebase seeding data

## 💡 **GỢI Ý THÊM**

- Hoặc tạo các hình ảnh với nhiều biến thể (ví dụ: dog_brown.png, dog_white.png, dog_black.png)
